# 📋 GEO優化檢查清單

這裡提供各種GEO優化場景的檢查清單，幫助您系統性地改善內容的AI引用潛力。

## 🎯 基礎GEO優化檢查清單

### ✅ 內容結構優化

**摘要段落：**
- [ ] 包含`<section class="summary">`標籤
- [ ] 摘要長度控制在150-300字
- [ ] 涵蓋文章核心關鍵詞
- [ ] 提供明確的價值主張
- [ ] 開門見山回答主要問題

**標題階層：**
- [ ] 使用H1作為文章主標題
- [ ] H2標題包含目標關鍵詞
- [ ] H3標題採用問句形式
- [ ] 標題階層邏輯清晰
- [ ] 每個標題下內容聚焦單一主題

**段落設計：**
- [ ] 每段控制在3-5句話
- [ ] 開頭句點出段落重點
- [ ] 避免過長的複合句
- [ ] 段落間使用過渡語句
- [ ] 每段可獨立理解

### ✅ 語意優化要素

**語意錨定：**
- [ ] 主題明確不發散
- [ ] 關鍵概念定義清楚
- [ ] 避免模糊或籠統的表達
- [ ] 使用具體的數據和例子
- [ ] 專業術語有適當解釋

**語境觸發：**
- [ ] 包含關鍵詞的同義詞變體
- [ ] 涵蓋不同的提問方式
- [ ] 使用相關的長尾關鍵詞
- [ ] 考慮用戶搜尋意圖
- [ ] 支援多種查詢表達

**語用重組：**
- [ ] 內容採用模組化結構
- [ ] 提供條列或編號清單
- [ ] 包含FAQ問答格式
- [ ] 每個模組可獨立引用
- [ ] 便於AI拆解重組

### ✅ 權威性建立

**資料來源：**
- [ ] 引用權威的官方文件
- [ ] 提供具體的統計數據
- [ ] 標註資料來源和日期
- [ ] 使用最新的研究報告
- [ ] 避免無根據的主觀判斷

**專業度展現：**
- [ ] 使用行業標準術語
- [ ] 提供實戰經驗分享
- [ ] 展示深度專業知識
- [ ] 包含獨特觀點或洞察
- [ ] 避免常見的初級錯誤

## 🚀 快速自檢清單

### ⚡ 5分鐘快速檢查

針對單篇文章的快速品質檢查：

- [ ] **摘要完整**：有summary段落且內容精煉
- [ ] **結構清晰**：H2/H3標題邏輯分明
- [ ] **FAQ存在**：至少包含3個問答
- [ ] **內鏈合理**：有2-5個相關內部連結
- [ ] **引用權威**：提及可靠的資料來源
- [ ] **行動導向**：結尾有明確的下一步建議
- [ ] **語法正確**：無明顯錯字或語法錯誤
- [ ] **移動友好**：在手機上閱讀體驗良好

## 🎯 使用說明

### 📖 如何使用這些檢查清單

1. **選擇適合的清單**：根據您的需求選擇對應的檢查清單
2. **逐項檢查**：按順序檢查每個項目，確實完成才打勾
3. **記錄問題**：發現問題時記錄具體狀況和改善方法
4. **追蹤進度**：定期回顧檢查結果，持續改善
5. **團隊共享**：與團隊成員分享清單，建立標準流程

### 🔧 客製化建議

根據您的具體情況，可以：
- **調整優先序**：根據資源和目標調整檢查項目順序
- **添加特定項目**：針對特殊需求增加專屬檢查項目
- **設定評分標準**：為每個項目設定量化的評估標準
- **建立模板**：將常用的檢查清單製作成可重複使用的模板
