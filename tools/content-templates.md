# 📝 AIOGEO內容模板庫

這裡提供各種GEO優化文章的寫作模板，幫助您快速創建高品質、AI友善的內容。

## 📋 基本文章模板

### 🏗️ 標準GEO文章結構

```markdown
# 文章標題：包含主要關鍵字的描述性標題

<section class="summary">
摘要段落：150-300字的內容精華，涵蓋核心結論和關鍵詞。
此區將用於SEO描述摘要，亦方便AI模型快速抓取內容概要。
包含1-2組目標關鍵詞，並點出文章的主要價值。
</section>

## 🎯 引言：建立背景與價值主張

簡述文章背景、目的和重要性。點出使用者痛點或核心問題，
並強調本文將提供的解決方案。篇幅建議1～2段。

## 📚 主要內容

### H2標題一：包含目標關鍵字的段落標題

#### H3具體問題：什麼是XXX？

開門見山回答問題，第一句直接給出答案。
後續2-3句補充細節說明，確保段落主題集中。

**實戰要點：**
- 要點一：具體的操作建議
- 要點二：相關注意事項
- 要點三：常見錯誤避免

### H2標題二：第二個重要主題

內容結構同上，每段保持3-5句的適中長度。

## ❓ 常見問題FAQ

**Q1: 最常見的問題？**  
A1: 簡要而直接的回答，控制在2-3句話內。

**Q2: 進階問題？**
A2: 提供詳細解答，可引用數據或專業來源。

## 🚀 總結與行動建議

重點摘要和實用的下一步指導。

## 📚 延伸閱讀
- [相關文章連結](link1.md)
- [工具推薦](link2.md)
```

1. **保持一致性**：使用相同的風格和格式
2. **適應主題**：根據內容調整模板結構
3. **加入個性**：在模板基礎上添加獨特元素
4. **持續優化**：根據讀者反饋改善模板
